## Description
1-2 sentence description of what it will do and how it relevant to media and digital arts.
A visual program that simulates an ecosystem based on some basic rules that creates a random wallpaper every time.
This program is relevant to media because it is a wallpaper creator that utilizes randomness and code to create art.

## Features
- Feature 1: A cell sandbox where an image is randomly generated when the program is run.
	- The main feature where cells grow and interact with other cells in order to create visual art that is unique every time.
- Feature 2: When the user clicks on the screen, colors are added onto the screen at that point randomly.
	- Clicking on the screen causes colors to sprout from the click point in order to add visual flair
- Feature 3: Alternate versions with different base rules for differnt results.
	- This is an alternate option for the user to choose that has different base rules for the cells to follow that give different final results.
