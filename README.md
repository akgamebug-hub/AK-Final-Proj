## Description
    This program is a ecosystem simulator that has 3 different settings indexed by the space bar. When starting the program a title screen with controls will show up. Press enter to start the program and click on the screen to add some random splotches and change the color on the screen. Pressing Esc will exit the program. Thank you
    
    Included files:
    src/program.py # This is the .py file with all the code.
    requirements.txt # includes the 3rd party libraries that will need to be installed for the program to run.
    README.md # Includes repo URL, Youtube URL, and description.
